# Lisbon
Autonomous Security Agent

An AI-powered security agent that automatically audits and fixes vulnerabilities on your PC.

## Features
- ✅ 20+ security audit checks
- 🔧 Automatic vulnerability fixing
- 📊 Detailed security reports
- 🚨 Real-time monitoring
- 🎯 Risk score calculation
- 📝 Comprehensive logging

## Quick Start

### Installation
```bash
git clone https://github.com/yourusername/security-agent.git
cd security-agent
pip install -r requirements.txt


# Quick audit
python src/agent.py --audit

# Auto-fix all issues
python src/agent.py --fix

# Continuous monitoring
python src/agent.py --monitor
